# terminal-clock
A terminal clock

## Requirements
- Python3
- curses (A python curses libary)
- A terminal (Obivious)

## How to use 
​	Use your terminal & just `python ./clock.py`.

​	Press `q` to quit.

​	Clock will read your terminal size. Two size font available.
